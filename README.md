# 🧠 Sentiment Analysis using AI/ML

## Overview

This project focuses on performing sentiment analysis on Twitter data using multiple machine learning and deep learning models. It demonstrates data preprocessing, feature extraction using TF-IDF, and comparative evaluation of different classification algorithms.

## Features

- ✅ Cleans and preprocesses raw Twitter text data
- ✅ Encodes categorical sentiment labels into numerical format
- ✅ Uses TF-IDF with n-grams for feature extraction
- ✅ Implements multiple models:
  - Logistic Regression
  - Random Forest
  - Feedforward Neural Network (TensorFlow)
- ✅ Evaluates models using accuracy, precision, recall, and F1-score
- ✅ Generates confusion matrices and training performance plots
